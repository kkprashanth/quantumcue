# Example Data for Demo

This directory contains example files for testing and demonstration purposes.

## Brain Scan Image

**File:** `brain_scan_example.png` (or `.jpg`)

This is a grayscale MRI brain scan image suitable for testing the Alzheimer's detection model. The image shows:
- Axial/horizontal view of a human brain
- Clear anatomical structures (gray matter, white matter, ventricles)
- High contrast, diagnostic-quality image
- Suitable for quantum neural network classification

### Usage

When testing the "Alzheimer's Detection QNN - QCI Photonic" model:

1. Navigate to Model Detail page → Test Model tab
2. Click "Choose File" or drag-and-drop
3. Select `examples/data/brain_scan_example.png`
4. Click "Upload & Predict"
5. Review the prediction results

### Image Specifications

- **Type:** MRI brain scan (axial view)
- **Format:** PNG or JPG
- **Recommended size:** 256x256 pixels (will be resized if needed)
- **Content:** Grayscale medical imaging
- **Use case:** Testing quantum ML model inference

---

**Note:** Add your brain scan image file to this directory. The image should be a valid MRI brain scan for best results with the Alzheimer's detection model.
