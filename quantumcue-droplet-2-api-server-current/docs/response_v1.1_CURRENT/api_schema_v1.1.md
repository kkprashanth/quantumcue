# ML Metrics & Curves API — Schema Documentation (v1.1)

**Version:** 1.1  
**Last Updated:** 2026-02-22  
**Transport:** FastAPI / REST  
**Format:** JSON  

---

## Overview

This API returns synthetic (debug) and real ML evaluation data for multi-class binary classification models. Each response is identified by a unique `job_id` and contains:

- **Scalar summary metrics** per class + **macro-averages** (Accuracy, Precision, AUC-ROC, AUC-PR, Max F1)
- **Curve data** per class + **macro-averaged curves** (F1 vs Threshold, ROC, PR, Confusion Rates vs Threshold)

**NEW in v1.1:** Averages enable single-line dashboard plots alongside per-class curves.

Responses generated by Python modules in `modules/`:

| Module | Responsibility |
|--------|----------------|
| `metrics_generator.py` | Scalar metric arrays |
| `f1_generator.py` | F1 vs threshold curves |
| `roc_pr_generator.py` | ROC + PR curves |
| `confusion_generator.py` | TPR/FPR/TNR/FNR vs threshold |

**NEW:** `pipeline.py` computes all macro-averages (arithmetic means for shared axes; interpolated macro-averages for ROC/PR).

## Field Reference

### Root Fields
| Field | Type | Description |
|-------|------|-------------|
| `job_id` | `string` | `job_YYYYMMDD_<hash>` |
| `created_at` | `string` | ISO 8601 UTC |
| `num_classes` | `int` | 2–10 |
| `num_points` | `int` | 20–100 (curves); 20 for confusion |

### `metrics` Object
**`per_class`** (v1.0): Arrays length=`num_classes`, [0.80, 0.95].  
**`averages`** (**NEW v1.1**): 5 scalars = `np.mean(per_class[name])`, 4 decimals.

| Field | Description |
|-------|-------------|
| `accuracy` | Overall accuracy |
| `precision` | Macro precision |
| `auc_roc` | ROC area |
| `auc_pr` | PR area |
| `max_f1` | Peak F1 over thresholds |

### `curves.f1` (**UPDATED**)
| Field | Type | Shape | Description |
|-------|------|-------|-------------|
| `thresholds` | `float[]` | `num_points` | Shared x-axis [0.0, 1.0] |
| `per_class` | `float[][]` | `[num_classes][num_points]` | F1 curves |
| `average` (**NEW**) | `float[]` | `num_points` | `np.mean(per_class, axis=0)` |

**Behavior:** Starts ~0.10, peaks ~0.75 at threshold≈0.45, ends ~0.20.

### `curves.roc` (Unchanged)
Array length=`num_classes`. **Each** has independent `fpr`/`tpr` arrays (`num_points`, [0.0, 1.0]).  
Starts (0,0), steep near FPR=0, ends (1,1). Passes ~(0.10, 0.94).

### `curves.pr` (Unchanged)
Array length=`num_classes`. **Each** has independent `recall`/`precision` arrays (`num_points`, [0.0, 1.0]).  
Starts (0, 1.0), high then sharp drop, ends (1, 0).

### `curves.confusion_rates` (**UPDATED**)
**`thresholds`**: Shared x-axis (`num_points`, [0.0, 1.0]).  
**`per_class`**: 4 matrices `[num_classes][num_points]`.  
**`averages`** (**NEW**): 4 arrays (`num_points`) = `np.mean(per_class, axis=0)`.

**Relationships:** `tnr = 1-fpr`, `fnr = 1-tpr`.

### `curves.averages` (**NEW TOP-LEVEL**)
**ROC macro-average:** `fpr` = `np.linspace(0,1,num_points)`; `tpr` = interpolated macro-average.  
**PR macro-average:** `recall` = `np.linspace(0,1,num_points)`; `precision` = interpolated macro-average.

---

## Design Decisions

| Decision | Rationale |
|----------|-----------|
| `roc`/`pr` = array of objects | Independent x-axes per class |
| `f1`/confusion = shared x + matrix | Compact for shared thresholds |
| **NEW** `per_class` + `averages` pattern | Clear separation; backward-compatible |
| **ROC/PR** averages at `curves.averages` | Avoids breaking array iteration; standard macro-averaging needs interpolation |
| Floats to 4 decimals | Compact JSON |
| `job_id` root-level | Caching/deduplication |

**v1.1 Breaking Changes:** None. New fields ignored by v1.0 clients.

## Implementation Summary

**`pipeline.build_ml_result_payload(job_id, num_classes=4)`** orchestrates:

```
1. generators → raw per-class data
2. compute_averages() → all macro-averages (direct mean + interpolated ROC/PR)
3. assemble → v1.1 JSON (all .tolist())
```

**Usage in `main.py` v0 endpoint:**
```python
if random.random() < 0.5:
    return build_ml_result_payload(job_id=job_id, num_classes=num_classes)
```

## Client Notes

**Plotting v1.1:**
```javascript
// Dashboard: 1 average line + per-class lines
plot(curves.f1.average, curves.f1.thresholds, "avg F1")
for (let cls of curves.f1.per_class) plot(cls, curves.f1.thresholds, `class ${i}`)

// ROC macro-average (pre-computed!)
plot(curves.averages.roc.tpr, curves.averages.roc.fpr, "macro ROC")
plotROC(curves.roc[0]); // class 0
```
